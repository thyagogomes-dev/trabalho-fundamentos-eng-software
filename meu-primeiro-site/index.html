O que é Engenharia de Software?

Diferente da simples programação, a Engenharia de Software foca na aplicação de métodos sistemáticos, disciplinados e quantificáveis para o desenvolvimento e manutenção de sistemas de software complexos.

  • Qualidade: Garantir que o produto faça o que deve fazer.
  • Processo: Seguir etapas para evitar caos.
  • Manutenibilidade: Escrever código que outros (ou você no futuro) consigam ler.

Metodologia Ágil: SCRUM

O Scrum é um framework ágil usado para gerenciar projetos complexos. Ele divide o trabalho em ciclos chamados Sprints.

Papéis Principais:

Product Owner: Define o que deve ser feito.

Scrum Master: Ajuda o time a seguir o processo.

Developers: Quem realmente constrói o software.
